A Gutenberg block for creating a Hero Banner with slider options and background video support.

== Description ==

**Hero Banner Block** is a custom Gutenberg block that allows you to create modern hero sections with:

- Image or video background  
- Slider support for multiple banners  
- Overlay text, headings, buttons  
- Customizable height, alignment, and styling  

Ideal for homepages, landing pages, and promotional sections.

== Features ==

- Background image or video
- Slider with multiple slides
- Customizable overlay content
- Responsive design
- Gutenberg block editor support
- Lightweight and optimized

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.  
2. Activate the plugin through the **Plugins â†’ Installed Plugins** menu.  
3. Open the Block Editor and search for **Hero Banner Block**.  
4. Add it to any post or page and configure the settings.

== Frequently Asked Questions ==

= Does it work with any theme? =
Yes, it works with any theme that supports Gutenberg blocks.

= Can I use a YouTube or Vimeo URL for the video background? =
Currently, only self-hosted MP4 videos are supported.
